# HAVEN Game Assets

Place the following CC0 Kenney assets in these directories:

## game/public/assets/bg/
- `village.png` — from Kenney Tiny Town pack (https://kenney.nl/assets/tiny-town)
  Use as the village background (960×640 or similar)

## game/public/assets/sprites/
- `player.png`  — 16×16 top-down character spritesheet (4 rows × 3 frames = walk UDLR + idle)
- `pip.png`     — Fox sprite, 16×16, 2 frames for idle animation
- `bramble.png` — Bear sprite, 16×16, 2 frames
- `flint.png`   — Wolf sprite, 16×16, 2 frames
- `luna.png`    — Owl sprite, 16×16, 2 frames
- `cleo.png`    — Rabbit sprite, 16×16, 2 frames

All character sprites from: https://kenney.nl/assets/animal-pack-redux

## game/public/assets/ui/
- `speech-bubble.png` — small speech bubble icon (32×32 or similar)
- `panel-ninepatch.png` — chat panel background for NinePatch (Kenney UI Pack)

## game/public/assets/audio/
- `village-ambient.mp3` — gentle ambient loop (freesound.org, CC0)
- `chat-open.wav`       — soft chime sound
- `step.wav`            — soft footstep sound

## Asset download links
- Tiny Town: https://kenney.nl/assets/tiny-town
- Animal Pack Redux: https://kenney.nl/assets/animal-pack-redux
- UI Pack: https://kenney.nl/assets/ui-pack
- Sounds: https://sfxr.me or https://freesound.org (filter CC0)
